Instructions pour payer

  1. Sélectionnez le pack d'UC que vous souhaitez acheter
  2. Notez le montant exact en Ariary
  3. Effectuez le paiement via Mvola ou Orange Money vers les numéros indiqués
  4. Envoyez le montant exact (ne changez pas le montant)
  5. Remplissez le formulaire de confirmation avec votre ID PUBG et le numéro utilisé pour payer
  6. Joignez une capture d'écran du paiement si possible
  7. Votre UC sera ajouté à votre compte dans les plus brefs délais

Astuce : Votre ID PUBG se trouve dans votre profil de jeu. C'est une chaîne de chiffres unique à votre compte.
